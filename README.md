# HTMLToJSON
Flatten your html source, and generate json text data

## What is Used To be

If we have some html code
```html
<div>
    <p>helloworld</p>
    <span>
        text
        <br />
        text2
        <br />
        <span>text3</span>
        <span>text4</span>
    </span>
    <p>123456<span>78</span>90<br/><span>2333333</span></p>
</div>
wwwww
<span>12323</span>
1232323
213123123
<br />
```
